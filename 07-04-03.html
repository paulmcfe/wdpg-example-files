An Overly-Fussy List of CSS Class Name Restrictions

  • The class name can begin with a letter, underscore (_), or hyphen (-).
  • Class names beginning with an underscore or hyphen are typically reserved for browser-specific extensions.
  • If you do use a hyphen, the second character must be a letter or underscore.
  • The rest of the class name can use any combination of letters, numbers, hyphens, or underscores.
  • The name must be at least two characters long.
  • Technically, you can use special characters in the class attribute, but to refer to a special character in your CSS class name you must “escape” the character by preceding it with a forward slash (\). For example, if you use class="this&that"> in your HTML, your CSS class must be .this\&that.
