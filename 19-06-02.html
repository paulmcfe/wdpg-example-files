The Limits of Inheritance

Why don’t all CSS properties inherit their parent’s styles*? Because in some cases it would simply lead to weird or nonsensical results. For example, if you apply a border around, say, a div element, it would look odd indeed to apply the same border to a child span or strong element. Similarly, applying, say, a p element’s width value to a child em element doesn’t make sense.
* This W3C page tells you which styles don’t inherit.
